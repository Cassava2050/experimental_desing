# Check trial experimental desing

This repo contains the code needed for check experimental design data of trials
carried out in: 
- Palmira, Colombia
- North cost, Colombia
- South east, Asia

## Functions

- row_col_dup()
-- Checks for duplicates in rows and columns
- plot_number_dup()
-- Checks for duplicates in plot_number column
